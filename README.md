## 🛠️ Built With

This project is a modern, static web application built using:
- **HTML5** for semantic markup and structure
- **CSS3 / Vanilla CSS** for custom glassmorphism styles, fluid typography, and complex animated shapes
- **Vanilla JavaScript** (ES6+) for smooth scrolling, interactive navigation, and form handling
- **FontAwesome** for scalable vector icons
- **Google Fonts** (Outfit & Inter) for clean typography
- **FormSubmit** for handling backend contact form submissions without a dedicated server

## ✨ Key Features

- **Glassmorphism Design:** A modern, beautiful aesthetic featuring frosted glass elements, translucent backgrounds, and vibrant gradients.
- **Micro-Animations:** Fluid CSS transitions, floating background shapes with blur effects, and on-scroll fade-in delays.
- **Fully Responsive Layout:** Optimized across desktop, tablet, and mobile breakpoints with CSS Grid and Flexbox.
- **SEO Optimized:** Complete meta tags, Open Graph properties for social sharing, and semantic HTML structure.
- **Form Integration:** A fully functional, serverless contact form ready for production.
